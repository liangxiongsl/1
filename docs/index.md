10. 多机调度问题（填空-根据问题，能给出在各台机器上处理作业的总时间）p274

    问题：有n个作业分配给m台机器执行；作业i有处理时间 t_i；最小化总处理时间

    贪心：对处理时间降序排序；开一个优先队列(小根堆)，每次从堆中取最小值x，x累加t_i，x再放回堆中；堆的最大值 = 总处理时间
