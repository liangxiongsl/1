```title="4.资本主义的本质&&规律" linenums="1"
马政经：研究生产关系&&发展规律
马政经-创立：《资本论》
马政经~英古政经-区别：前者区分了 具体劳动 和 抽象劳动 (劳动二重性)

自然经济：自给自足
商品经济：用于 交换
商品经济-历史条件：生产资料 和 劳动产品 属于不同的所有者；存在社会分工
商品经济-基本矛盾：私人劳动 与 社会劳动 的矛盾
商品经济-基本矛盾-解决：交换
商品经济-阶段：简单商品经济，发达商品经济
商品经济-简单x：生产资料私有制，个体劳动
商品经济-发达x：
自然经济&商品经济-区别：生产目的不同


商品：是 劳动产品，用于 交换
商品-二因素：使用价值，价值  (不可兼得，要获得其一就必须放弃另一个)
商品-使用价值：人与自然之间的物质关系
商品-使用价值-性质：有用性，自然属性
商品-价值：制造商品中耗费的体力/脑力劳动
商品-价值-性质：社会/特有/本质 属性
商品-价值量：由 生产商品所耗费的劳动量 决定
商品-耗费的劳动量：由 劳动耗费的时间 计量

劳动-二重性：具体劳动，抽象劳动
劳动-具体劳动：形成使用价值
劳动-抽象劳动：形成价值
劳动力：人的劳动能力，人的脑力&体力的总和
劳动力-价值：劳动力 本人 / 家属 生存所必需的生活资料的价值，劳动者接收教育和训练所支出的费用
劳动力成为商品：从 简单商品生产 发展到 资本商品生产
劳动力成为商品-基本条件：劳动者 在法律上是自由人 / 没有任何生产资料
劳动力商品-突出特点：其 使用价值 是 价值 的源泉

商品-价值量-决定因素：社会必要劳动时间
价值规律：商品价值量 有 社会必要劳动时间决定，商品交换 以价值量为基础 / 按照等价交换原则进行
价值规律-作用：自发地 控制生产资料和劳动力在社会各生产部之间的分配比例 / 刺激社会生产力发展 / 调节社会收入分配
价值规律-调节经济活动-消极后果：导致 社会资源浪费，阻碍 技术进步，导致 收入两极分化

价格：受 市场供求 影响
价格&价值：上下波动的价格的平均数 与 价值 相一致

货币：一般等价物
货币-职能：基本/衍生 职能
货币-基本职能：价值尺度(标价)，流通手段(一手交钱一手交货)
货币-衍生职能：支付手段(提前/延后支付)，储藏手段(金属货币/条块)，世界货币(美元，英镑...)


剩余价值：超过雇佣工人所创造而被资本无偿占有的那部分价值
剩余价值-分类：绝对剩余价值(加班)，相对剩余价值(提高社会必要劳动生产率)
剩余价值-绝对xx：在必要劳动时间不变的条件下, 由于延长工作日的长度和提高劳动强度而生产的剩余价值
剩余价值-相对xx：在工作日长度不变的条件下, 通过缩短必要劳动时间而相对延长剩余劳动时间所生产的剩余价值 (即提高)
剩余价值-来源：工人的剩余劳动
剩余价值率：m'=m/v (剩余价值率 = 剩余价值 / 可变资本)
剩余价值率-作用：确认资本家对工人的剥削程度
剩余价值-有机构成：c : v


资本-分类：可变资本(v)，不变/固定资本(c)
资本-可变xx：购买劳动力花费的资本？工资？
资本-不变xx：以生产资料的形式存在的资本？
资本运动-资本总公式：货币/价值 -> 商品 -> 增值后的货币/价值
资本积累：资本主义事业现象的根源
资本循环：资本从一种形式出发，经过一系列形式的变化，又回到原来的出发点的运动

货币-本质：体现 社会关系
利润：资本家眼中的 剩余价值
利润率：预付资本的增值程度 (利润率 = 剩余价值 / 全部预付资本)
工资：劳动力价值/价格的货币表现
工作日：必要劳动时间 + 剩余劳动时间


资本积累：剩余价值资本化
资本积累-目的：榨取更多剩余价值
资本积累-规模-依赖于：对工人的剥削程度，劳动生产率的高低，所用资本和所费资本之间的差额，资本家预付资本的大小
资本积累-结果：社会财富 集中在少数人手里 / 两极分化(马太效应)，产生社会失业现象
资本主义原始积累-途径：暴力剥夺农民土地，暴力掠夺货币财富

资本主义-产生原因：由小商品经济分化而来，从商人和高利贷转化而来
资本主义-基本矛盾：生产资料私人占有 和 生产社会化 的矛盾
资本主义-经济制度：生产资料私有制 -> 劳动力成为商品 (劳动力是自由人/一无所有)
资本主义-生产过程：生产资料价值 通过工人的具体劳动转移到产品中 / 不发生增值
资本主义-生产过程-二重性：既是 劳动过程 又是 价值增值过程
资本主义-生产过程-特点：劳动成果/产品全部归资本家所有
资本主义-经济危机-根本原因：资本主义的基本矛盾
资本主义-经济危机-本质特征：生产过剩 (与 劳动人民的支付能力需要 相关)  
资本主义-选举-实质：协调统治阶级内部利益关系和矛盾的重要措施
资本主义-法律体系-支柱：私有制
资本主义-民主：金钱操纵下的民主
资本主义-政党制度-实质：资产阶级选择自己的国家管理者，实现内部利益平衡的政治统治

资产阶级-意识形态-核心：政治/法律 思想 

资本主义国家-职能-根本内容：服务于 资本主义制度 和 资产阶级利益/统治
资本主义国家-职能-鲜明体现：政治统治职能
资本主义国家-政权组织形式：分权制衡
资本主义国家-法律：服务于 资产阶级特权

劳动产品-共有属性：使用价值
社会财富-物质内容：使用价值


交换价值：一种使用价值与另一种使用价值相交换的量的比例

“惊险跳跃”：商品转化为货币

生产者与生产资料-分离的手段：暴力手段


经济全球化-表现：生产全球化(核心)，贸易全球化，资本全球化
经济全球化-实质：发达资本主义国家主导
经济全球化-积极作用：全球资源有效配置，全球合作

所有制&所有权-关系：所有制 是 所有权 的基础，所有权 是 所有制 的法律形态
```

```title="5.资本主义发展&&趋势" linenums="1"
资本主义-发展阶段：自由竞争资本主义，垄断资本主义
资本主义-变化的根本推动力：生产力的发展和科学技术革命
资本主义-基本经济制度：生产资料私有制

生产集中：生产资料，劳动力，商品的生产 日益集中于少数大企业的过程
资本集中：大资本吞并小资本 / 许多小资本合并成大资本

垄断-产生原因：生产集中发展到相当高的程度，企业联合；企业规模巨大，形成对竞争的限制；激烈的竞争给竞争各方带来越来越严重的损失，为避免两败俱伤，企业间会达成妥
垄断-特点：采取非经济手段竞争，竞争范围扩大到经济外
垄断-形式：垄断价格
垄断-主要目的：获取垄断利润
垄断-商品-出售方式：垄断价格？
垄断利润：垄断资本家通过社会生产和流通中的垄断地位而获得的 高于平均利润 的高额利润
垄断利润-根本来源：劳动工人创造的剩余价值

垄断资本-向世界扩展-基本形式：借贷/生产/商品 资本输出

私人垄断资本主义-基础：生产集中，资本集中
国家垄断资本主义-实质：私人垄断资本主义利用 国家机器 为其发展服务
国家垄断资本主义-形成原因：社会生产力发展，要求资本主义生产资料在更大范围内被支配；经济波动&经济危机 深化；缓和社会矛盾；协调利益关系
国家垄断资本主义-主要形式：微观调制，宏观调制，。。。
国家垄断资本主义-微观调制-主要类型：反托拉斯法，公共事业规制，社会经济规制

金融寡头：操纵国民经济命脉，并实际操制国家政权的 少数垄断资本家/集团
金融寡头-经济领域-统治的实现：参与制
金融资本-壮大的制度条件：金融自由化，金融创新
金融垄断资本-作用：促进资本主义经济发展，造成经济过度虚拟化


经济全球化：生产/贸易/金融 全球化
经济全球化-特点：不可阻挡的历史必然性
经济全球化-物质基础/根本推力：科学技术 / 生产力 的发展

国家资本所有制：体现着总资本家剥削雇佣劳动者的关系

社会主义不能在所有国家同时获得胜利？
国际经济组织：国际货币基金组织，世界银行，世界贸易组织
资本家-鼓励制度：职工参与决策，终身雇佣，职工持股
二战后的资本主义的变化-原因：人类社会发展一般规律，资本主义经济规律
```
